O que é um elemento?

é uma parte de uma página web que pode conter vários atributos. Estes podem ser dados, texto ou imagem. normalmente inicia com uma tag, seguido do atributo e finalizando com uma tag de novo

Atribuição

A atribuição é posta a seguir da abertura das tags é onde se atribui as especificações do elemento.
